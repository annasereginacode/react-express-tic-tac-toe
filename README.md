# Server-Backed Tic-Tac-Toe

A full-stack Tic-Tac-Toe game built with React, Vite, Express, and JSON file persistence.

## Features

- Tic-Tac-Toe gameplay
- Move history and time travel
- Winner detection
- Draw detection
- Reset game button
- Game state automatically saved to the server
- Game state automatically restored after page refresh

## Running the Server

Open a terminal in the `server` folder:

```bash
npm install express
node index.js
```

## Running the Client

Open a second terminal in the `client` folder:

```bash
npm install
npm run dev
```

## Open the Application

Open the following URL in a browser:

```text
http://localhost:5173
```

## Technologies Used

- React
- Vite
- Express
